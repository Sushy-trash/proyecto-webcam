- Instalar y ejecucion

Tines que tener Python 3.11 instalado. Una vez verificado, instala las dependencias necesarias con el gestor de paquetes pip:

-Instalación de las librerías principales

pip install opencv-python
pip install mediapipe==0.10.11


- Archivos dispensables del Proyecto
Asegúrate de que tu carpeta se vea así antes de empezar:

mainb.py (El código fuente)
image.png (La imagen de "Es cine")
like.png (La imagen de "No era png")
giphy.gif (El GIF del gato)


- Ejecución

Para iniciar el programa, navega hasta la carpeta del proyecto y utiliza el comando:

python mainb.py
